How It Works

The inspector adds data-source-file and data-source-line attributes to your HTML elements during a build step. When you hover or click elements, it shows you:

  • Source location (file and line)
  • CSS selector path
  • Computed styles
  • DOM hierarchy
  • Dimensions and layout

Quick Start

Add the inspector to your HTML:

<link rel="stylesheet" href="/inspector/element-inspector.css"> <script src="/inspector/element-inspector.js"></script>

Then annotate your HTML files:

node annotate-html.js input-dir output-dir
